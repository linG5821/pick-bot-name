## 📝 文档标准

本仓库中的所有文档遵循以下约定：

- **格式**：GitHub 风格的 Markdown
- **结构**：清晰的标题（H1 → H6）
- **示例**：带语法高亮的代码块
- **链接**：其他文档的相对链接
- **语言**：英文（主要），适用时有中文示例
- **更新**：对时间敏感的内容包含"最后更新"日期

## 🤝 为文档做贡献

发现错误或想改进文档？

1. **小修正**：直接提交 PR
2. **大改动**：先打开 issue 讨论
3. **新部分**：遵循现有结构和风格
4. **代码示例**：提交前测试它们
