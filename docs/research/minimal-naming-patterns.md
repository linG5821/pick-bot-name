## Naming Pattern Analysis

### Chinese Patterns

**1-Character Names:**
- Often abstract concepts (云, 信, 智)
- Single-syllable, memorable
- Common in tech domain names
- Used as prefixes in longer names
- 20% of minimal Chinese names

**2-Character Names:**
- Dominant pattern (80% of market)
- Natural Chinese word length
- Often reduplication (钉钉, 滴滴, 球球)
- Mix modern + traditional characters
- Easy to remember and say
- WeChat mini programs favor this

**Key Principles:**
- Reduplication adds friendliness (萌萌, 团团)
- Tech terms blend EN + CN (云端, 极客)
- Avoid 的 (possessive) structure
- Prefer existing words over invented compounds
- Sound matters: avoid awkward pronunciations

### English Patterns

**Single Word (60% of minimal names):**
- Tech terms (Hub, Core, Node, Edge)
- Descriptive adjectives (Quick, Smart, Pure)
- Action verbs (Link, Flow, Spark)
- Natural nouns (Bear, Things, Arc)
- 4-6 letters ideal length

**Two Words (40% of minimal names):**
- Adjective + Noun (Smart Hub, Quick Note)
- Action + Noun (Clean Code, Track Time)
- Tech + Term (Cloud Core, Data Hub)
- Descriptor + Bot (Task Bot, Help Bot)

**Key Principles:**
- Pronounceable in one breath
- Avoid articles (The, A)
- Prefer strong verbs/nouns
- Tech terms signal modern/professional
- "-Bot" suffix instantly identifies purpose
- Compound words feel more feature-rich

## Recommendation for Bot Name Generator

### Chinese (2 chars ideal):**
Priority patterns:
1. Tech terms: 智能, 云端, 极简
2. Reduplication: 点点, 链链, 码码
3. Action + Noun: 快搜, 智答, 秒回
4. Cute duplication: 喵喵, 团团, 呆呆

**English (1-2 words):**
Priority patterns:
1. Tech + Bot: Hub Bot, Core Bot, Link Bot
2. Adjective + Bot: Quick Bot, Smart Bot, Pure Bot
3. Single tech terms: Nexus, Pulse, Echo, Spark
4. Action + Noun: Quick Help, Smart Task, Fast Link

**Cross-Language Insights:**
- Both prefer extreme brevity
- Tech terms universally recognized
- Reduplication (CN) = alliteration (EN) for friendliness
- Abstract concepts work across languages
- Avoid overly descriptive multi-word phrases
