## Pattern Analysis

### Chinese Patterns (2-4 chars)

**2-Character Core Patterns**:
1. **Self-deprecating humor**: 码农, 码畜, 民工 (humble/working class metaphors)
2. **Animal puns**: 程序猿 (ape), 码农 (peasant), 攻城狮 (lion/engineer)
3. **Work culture**: 摸鱼 (slacking), 划水 (floating), 搬砖 (grunt work)
4. **Status terms**: 大神 (god), 菜鸟 (newbie), 老鸟 (veteran)
5. **996 culture**: 修仙 (overtime as "cultivation"), 爆肝 (overwork)

**3-Character Enhancement**:
- Base + Suffix: 算法君 (Algorithm + kun), 测试喵 (Test + meow)
- Role + Modifier: 全栈侠 (fullstack + hero), 前端仔 (frontend + guy)
- Tech + Title: 架构师 (architect), 炼丹师 (alchemist/ML)

**4-Character Descriptive**:
- Skill + Mastery: 摸鱼达人 (slacking master), 调试专家 (debug expert)
- Tech + Metaphor: 内存刺客 (memory assassin), 线程杀手 (thread killer)
- Domain + Role: 代码诗人 (code poet), 性能杀手 (performance killer)

### English Patterns (1-4 words)

**Single Word**:
- Fantasy/mystical: ninja, wizard, guru, master
- Tech terms: byte, bit, pixel, daemon, kernel
- Underground: hacker, ghost, phantom, cipher

**2-Word Compounds**:
- [Tech] + [Role]: code ninja, debug master, api guru
- [Action] + [Object]: bug slayer, pixel pusher
- [Platform] + [Title]: git master, shell master
- Animal metaphor: code monkey

**3-4 Word Phrases**:
- Epic titles: "lord of code", "master of the stack"
- Descriptor phrases: "the one who debugs"
- Fantasy references: Game of Thrones style ("breaker of builds")

### Key Differentiators

**Chinese vs English**:
- Chinese: Compact (2-4 chars), pun-heavy, self-deprecating, work culture references
- English: Longer phrases, fantasy/mystical themes, action-oriented

**Real Platform Usage**:
- GitHub Chinese devs: Mix of pinyin + tech terms (qixing-jk)
- GitHub English devs: Compound words, real names, or single words
- V2EX/Juejin: More cultural memes, work-life references
- Stack Overflow: Professional, descriptive, less playful

## Recommendations for Rule Generation

### Chinese (2-4 chars)
- **2-char**: Focus on puns, animals, status, work culture
- **3-char**: Add suffixes (君/酱/侠/师), role modifiers
- **4-char**: Descriptive phrases, skill + mastery combinations
- **Avoid**: Long phrases with 的, formal role titles creating 5+ chars

### English (1-4 words)
- **1-word**: Tech terms, fantasy roles, underground themes
- **2-word**: Compound patterns (tech + role, action + object)
- **3-word**: Epic titles, fantasy references
- **4-word**: Descriptive phrases (use sparingly)

### Generation Algorithm
- **Combination** works best for Chinese (prefix + root + suffix with weights)
- **Template** works for English compound patterns
- Weight popular patterns higher (ninja, master, 君, 侠)
